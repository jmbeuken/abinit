ABINIT Bibliography

A selection of papers containing the theoretical ideas underlying the abinit code, and selected applications.

This file is far from being complete, and is not maintained day-to-day. Please read the book from Richard Martin, as well as the four general ABINIT papers for a more complete overview ...
